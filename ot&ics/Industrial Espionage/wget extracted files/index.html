Challenge Description

This service tests your ability to remember and recall data across requests. Your task is to:

  • Make requests to collect data and flag parts
  • Remember previous responses for memory questions
  • Answer memory questions correctly to unlock flag parts

Available Endpoints

  • /answer - Answer memory questions
  • /stats - Service statistics
